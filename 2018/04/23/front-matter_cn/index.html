Hexo参数

layout

布局

title

Title Filename (posts only)
title 标题 文章的文件名

data

Published date File created date
date 建立日期 文件建立日期

updated

Updated date File updated date
updated 更新日期 文件更新日期

comments

comments Enables comment feature for the post true
comments 开启文章的评论功能 true

tags

tags Tags (Not available for pages)
tags 标签（不适用于分页）

categories

categories Categories (Not available for pages)
categories 分类（不适用于分页）

permalink

permalink Overrides the default permalink of the post
permalink 覆盖文章网址

keywords

keywords The keywords that only used in meta tag and Open Graph (not recommended)
keywords 仅用于 meta 标签和 Open Graph 的关键词（不推荐使用）

excerpt

excerpt Page excerpt in plain text. Use this plugin to format the text
excerpt: 摘要（不需要在 Front-matter 中设置，通过 截断实现，预览卡片与正文中均出现）

Yun主题参数

type

= tags, categories, albums

为文章设置 type 属性，即可将其转为其他类型卡片，你也可以自己在 yun.yml 设置你跳转不同链接专属的图标和颜色。

type: tags
如果您尚未安装 hexo-generator-tag，请输入 npm install hexo-generator-tag。
type: categories
如果您尚未安装 hexo-generator-category，请输入 npm install hexo-generator-category。

type: albums
albums:

  • caption: 夕阳西下
    url: /albums/sunset.html
    cover: https://interactive-examples.mdn.mozilla.net/media/examples/elephant-660-480.jpg
    desc: 我想起那天夕阳下的奔跑

hide

hide: true

该文章仍然会被渲染，你自己可以直接访问链接进行查看。但不会被显示在展示的文章卡片中。

题外话，这个功能是我当初应付备案临时加的。 我更改备案信息时，客服通知我首页不能用跳转其他页面链接的内容（有一个和文章混在一起直接跳转 bilibili 的卡片），所以我就加了这个功能临时隐藏掉了。 也许还挺实用的，你可以放一些只是自己看看，暂时还不打算发到主页显示的页面。

值 内容 默认值 可选值 适用位置 说明
layout 页面布局 - - 页面
title 标题 自动生成 - 文章/页面/摘要
date 建立时间 自动生成 - 文章/页面/摘要
updated 更新时间 可自动生成 - 文章/页面/摘要
comments 是否开启评论 true true / false 文章/页面 默认不设置及跟随主题配置
tags 标签 - - 文章/页面/摘要
categories 分类 - - 文章/页面/摘要
permalink 连接 - - - 设定固定链接，没啥特殊要求别花时间研究它了
keywords 关键词 - - - 仅用于 meta 标签和 Open Graph 的关键词（不推荐使用）
author 作者 - - 文章/摘要
email 作者头像 - - 文章/摘要 仅在有 author 值时生效
description 描述 - - 文章/摘要
type 页面类别 - categories / tags / albums 页面
url 连接 文章/摘要 有type，文章无内容时，摘要直达链接
hide 隐藏 false true / false
icon 图标
reward 打赏 true true / false
