主题配置

安装

在 hexo 根目录下执行：

// 最新版
git clone -b dev https://github.com/YunYouJun/hexo-theme-yun themes/yun

// 稳定版
git clone -b master https://github.com/YunYouJun/hexo-theme-yun themes/yun

升级

在 yun 主题目录下执行：

git pull

插件

渲染器

npm install hexo-renderer-pug hexo-renderer-stylus

分类标签等系统插件

npm install hexo-generator-index hexo-generator-archive hexo-generator-category hexo-generator-tag

标签云

npm install hexo-tag-cloud

搜索引擎

npm install hexo-generator-search

主题配置参数

RSS

npm install hexo-generator-feed

代码高亮

npm install hexo-prism-plugin

根目录配置参数

字数统计

npm install hexo-symbols-count-time

根目录配置参数

主题配置参数

站点统计

npm install hexo-generator-sitemap hexo-generator-baidu-sitemap

配置原则

配置文件采用约定先于配置原则，即 root/source/_data/yun.yml 先于 themes/yun/_config.yml 。

出于升级及备份方便考虑，自定义素材均保存于 themes/yun/source/data/ 。
