This is the record of modification of theme diaspora.

Installation

git clone https://github.com/theme-next/hexo-theme-next themes/next

Required Plugins

Default plugins

npm install hexo-generator-index hexo-generator-archive hexo-generator-category hexo-generator-tag

RSS

npm install hexo-generator-feed

Sitemap

npm install hexo-generator-sitemap hexo-generator-baidu-sitemap

Time Counter

npm install hexo-symbols-count-time

Then, add configuration to root/_config.yml

Search

npm install hexo-generator-searchdb

Then, add configuration to root/_config.yml

Modificaitn

Post Card

File path: layout/_macro/post.swig

Footer

move the following block from _partials/footer.swig into _third-party/statistics/busuanzi-counter.swig with two lines changing the position.
